## 🧰 How to Use WriterClaw

WriterClaw has a simple interface split into three main parts:

- **Notes:** Write and keep your notes organized.
- **Drafts:** Work on documents with basic editing features.
- **Check:** Quickly check your writing for simple errors and suggestions.

### Writing Notes

- Click on the “Notes” tab.
- Use the large text box to write down your ideas.
- Save notes automatically or click the save button to keep them for later.

### Working on Drafts

- Select the “Drafts” tab.
- Start a new document or open an existing draft.
- Use the toolbar to format basic text: bold, italic, underline, and lists.

### Checking Your Writing

- Click the “Check” tab.
- Copy and paste your text or write directly inside the box.
- Click the “Check” button to find common mistakes.
- Suggestions will appear below the text box.
